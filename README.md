# DokjaTeka

## Descripción
Aplicación móvil desarrollada en Android que permite gestionar y visualizar contenido digital de forma organizada. La app utiliza una base de datos local para almacenar información, ofreciendo una experiencia funcional sin necesidad de conexión a internet.

## Funcionalidades
- Gestión y almacenamiento de datos de forma local
- Navegación entre diferentes vistas
- Interfaz de usuario enfocada en la usabilidad
- Persistencia de información mediante base de datos local

## Tecnologías
- Kotlin
- Android Studio
- SQLite

## Instalación
1. Clonar repositorio
2. Abrir en Android Studio
3. Ejecutar en emulador o dispositivo

## Objetivo del proyecto
Desarrollar una aplicación móvil funcional integrando almacenamiento local y buenas prácticas de desarrollo en Android.
